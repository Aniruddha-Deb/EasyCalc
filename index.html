EasyCalc

Calculator to make calculation easier, better and faster

View on GitHub Download .zip Download .tar.gz

EasyCalc is a simple, clean calculator made in Java. Contrary to it's name, EasyCalc has extremely advanced features which are located under the hood.

EasyCalc, unlike some calculators, has the ability to display the entire expression at one go. It has a clean and refined UI which adds to the functionality. It also integrates naturally with the keyboard of your computer, so that you don't always have to press down the buttons. To top it all off, EasyCalc as an extremely advanced parser which breaks down the code parsed into parse trees and then evaluates them via the BODMAS rule.
